# 關於康勞德亂碼

## 站點性質

非資工背景醫學生 vibe-coding + LLM workflow 觀察筆記。長文 deep-dive 為主，hedge-rich，不是 tutorial、不是 best practice 宣稱、是「我目前的做法是 X，還在調」這種 lab meeting 語氣。

## 作者

**康瑋麟（WLK）**

- 台大醫學系大六學生
- 沒上過任何 CS 系正規課程
- 透過 Claude Code 完成統計 / vibe-coding / skill 設計
- 主要使用情境：(1) 生物統計 / 臨床研究、(2) 醫學知識整理（國考準備）、(3) vibe-coding 業餘興趣

## 為什麼有這個站

我學 LLM workflow / Claude Code 一年多，散落產出在：

- **Threads** — 短文觀察（[`@weilingkang3`](https://www.threads.com/@weilingkang3)）
- **GitHub repos** — 一個 workflow 一個 repo
- **RemNote** — 私人筆記

中間少了一塊「長文 + 公開 + 對非 CS 背景同類人友善」的地方。本站補這塊。

## Voice 演化（A → B）

- **Phase A**（現在）：hedge-rich、個人語氣、學習日記。預設受眾是像我一樣**非 CS 背景但在 vibe-coding 的人**（醫學生、PhD 學生、研究助理）
- **Phase B**（未來）：帶 evidence 的 architecture observation。受眾擴到 AI engineer / Claude Code power user

每篇文章的 frontmatter 標 `phase: "A"` 或 `"B"`，作為自我觀察軸。
